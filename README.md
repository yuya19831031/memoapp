# メモ管理アプリ

メモの追加・検索・編集・削除ができる、シンプルなWebアプリです。

---

## 主な機能

- メモの新規作成
- メモ一覧表示
- タイトルによる検索
- メモ詳細表示
- メモ編集
- メモ削除
- 作成日・更新日の自動記録

---

## 作成した目的

Spring Bootの基本的な開発フローを理解するために作成しました。  
CRUD処理だけでなく、バリデーション、例外処理、ページング、検索機能など、  
実務で必要となる基本的な機能を一通り実装することを目的としています。

---

## 使用技術

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- H2 Database
- Maven

---

## 起動方法

./mvnw spring-boot:run

起動後、以下にアクセスしてください。

http://localhost:8086
